# Proyecto: Adivina el Número

### `Este proyecto es un sencillo juego web en el que el usuario debe adivinar un número aleatorio generado por el sistema. El juego proporciona pistas al usuario sobre si el número ingresado es mayor o menor que el número aleatorio.`

## Características
**Generación de un número aleatorio entre 1 y 100.** 

**Sistema de intentos ilimitados.**

**Pistas sobre si el número ingresado es mayor o menor que el número aleatorio.**

**Reinicio del juego con un solo clic.**

## Tecnologías Utilizadas
`HTML`

`CSS`

`JavaScript`

## Instrucciones de Uso
### **_1.-Clona el repositorio en tu máquina local._**

### **_2.-Abre el archivo index.html en un navegador web._**

### **_3.-Ingresa un número en el campo de texto y haz clic en el botón "Adivinar"._**

### **_4.-El sistema mostrará una pista sobre si el número ingresado es mayor o menor que el número aleatorio._**

### **_5.-Repite el paso 3 hasta adivinar el número._**

### **_6.-Haz clic en el botón "Reiniciar" para generar un nuevo número aleatorio._**


## Archivos

### **index.html: Archivo HTML que contiene la estructura y el contenido del juego.**

### **styles.css: Archivo CSS que contiene los estilos aplicados al juego.**

### **script.js: Archivo JavaScript que contiene la lógica del juego.**
